# Gazebo ROS workspace
This workspace is designed to build solutions for Gazebo and ROS environment combined using 'colcon'.

## Directory structure
* src
    * gazebo_dev
    * gazebo_msgs
    * gazebo_plugins
    * gazebo_ros
    * gazebo_ros_pkgs
